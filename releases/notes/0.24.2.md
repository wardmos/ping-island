## Compatibility
- Adds compatibility with the new Hooks flow in QoderWork `0.6.7+`.
- QoderWork `expectsResponse=false` tool notifications stay non-blocking and do not show approval prompts.

## Notes
- The Sparkle build number is now `62`, and the app version is now `0.24.2`.
- This release is based on latest `main` after `v0.24.1` and focuses on QoderWork `0.6.7+` Hooks compatibility.
