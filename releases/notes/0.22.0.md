## Highlights
- Added the official Pi Agent extension integration: Ping Island can now install the generated TypeScript extension and receive Pi events through the Claude-compatible bridge.
- Added Pi-specific client recognition, icon, colors, and mascot support so Pi sessions have consistent branding in the list, Notch, and settings previews.
- Remote SSH bootstrap now carries Pi hook configuration so remote Pi Agent sessions can return to the local Ping Island UI through the existing forwarding channel.

## Notes
- The Sparkle build number is now `57`, and the app version is now `0.22.0`.
- This latest-`main` release includes the Pi Agent extension integration from PR `#233`.
