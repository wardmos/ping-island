## Highlights
- Improved Agent recognition inside Qoder IDE terminals: when Claude or Claude-compatible sessions are started from Qoder IDE, Island more reliably separates the host IDE from the actual Agent and avoids labeling the session as Qoder.
- Added statistics entry points and overview data to Settings, making local Agent usage, token, and tool metrics easier to inspect.
- Reduced visual weight in the Settings window and Sound panel so phase sound controls align to the right and stay compact for repeated configuration work.

## Fixes
- Fixed Qoder IDE terminal context overriding the real Agent identity in some sessions.
- Fixed heavy or unstable alignment in compact Settings action rows and Sound controls.
- Added regression coverage for Settings category ordering so the Statistics entry stays high priority while Shortcuts remains later in the list.

## Notes
- The Sparkle build number is now `50`, and the app version is now `0.18.0`.
- This release focuses on Qoder IDE Agent recognition, Settings statistics, and Sound settings presentation polish.
