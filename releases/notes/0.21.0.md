## Highlights
- The floating pet can now be dragged back to the notch area to redock automatically, making the docked Island and detached pet transition smoother.
- Quiet background behavior now respects the current surface mode: a closed docked Island can hide completely while a detached pet remains visible at reduced opacity.
- Visible but long-settled idle sessions move to static frames after a 10 minute grace window, reducing unnecessary background animation work.
- Settings analytics now handle first-load state and oversized values more cleanly, avoiding empty-looking panels and cramped card overflow.

## Fixes
- Fixed remote SSH connection scripts being misinterpreted when the remote account uses fish or another non-POSIX shell, matching issue `#215`.
- Cleaned up Homebrew release docs and CI so the official Homebrew cask is the installation source instead of the legacy tap.
- Simplified the English and Chinese README install instructions to keep the legacy tap migration path out of the default flow.

## Notes
- The Sparkle build number is now `55`, and the app version is now `0.21.0`.
- This latest-`main` release focuses on floating pet redocking, remote SSH compatibility, settled low-power behavior, analytics display polish, and Homebrew release cleanup.
